---

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Tobyishiwu/tobyishiwu-portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
